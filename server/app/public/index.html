Heart Rate Monitoring

We have created a web application that monitors a users heart rate and blood oxygen saturation levels throughout the day. The app can be used by both patients and their physicians. The rate at which these measurements are taken can be configured by either party. This project is a culmination of our learnings through the semester, where we put together everything we have learned into one conducive project:

  • We use HTML, CSS, and JavaScript to create our website, format it, and make its components function.
  • We use Node.js(?), Express, and MongoDB to make our server function and store the collected data.
  • We use jQuery(?).
  • We use RESTful APIs(?).
  • We utilize password hashing and tokens to authenticate our users before allowing access to their account, as it contains sensitive medical information.
